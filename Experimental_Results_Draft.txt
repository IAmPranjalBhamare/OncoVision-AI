5.2. Formal Performance Metrics
The efficacy of the classification engine is measured through a multi-dimensional assessment of predictive accuracy. In a clinical context, the objective is to precisely categorize input ultrasound scans into targeted diagnostic groups (Benign, Malignant, or Normal). To quantify the model's discriminative ability, we utilize a Confusion Matrix comprising True Positives (TP), True Negatives (TN), False Positives (FP), and False Negatives (FN). The following performance indices are utilized:

- Accuracy (Overall Prediction Accuracy): This is the ratio of correctly predicted diagnostic instances (both positive and negative) to the total number of samples in the testing dataset. It provides a generalized view of the model's overall correctness.

  Accuracy = [TP + TN] / [TP + TN + FP + FN]

- Precision (Positive Predictive Value): This metric identifies the ratio of accurate malignant predictions relative to the total number of positive classifications. It is vital for assessing the reliability of malignancy alerts.
  
  Precision = TP / [TP + FP]

- Recall (Sensitivity): Recall measures the model's sensitivity in identifying all malignant cases from the ground-truth population. This is a critical metric for diagnostic safety, as it minimizes the risk of missed cancer detections.

  Recall = TP / [TP + FN]

- F1-Score: As the harmonic mean of precision and recall, the F1-score provides a singular measure of diagnostic robustness, particularly effective for handling class imbalances within the medical dataset.

  F1-score = 2 * [(Precision * Recall) / (Precision + Recall)]
